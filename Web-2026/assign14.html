Machine Learning-Driven Predictive Maintenance for Vehicles Using Audio Analysis
Created a reliable, affordable diagnostic system that uses engine acoustic data recorded by commonplace devices like smartphone microphones to change maintenance from reactive to predictive. The main innovation is a two-stage classification pipeline that uses a Random Forest (RF) model for conditional, particular fault identification and a Support Vector Machine (SVM) for high-confidence binary anomaly detection (reaching over 94% accuracy in anomaly detection). Using sophisticated acoustic features like MFCCs to provide objective, real-time diagnostic results, the solution was developed as a modular Python Flask web application.
